Cliquer pour afficher le panneau

Ce panneau contient un élément div, qui est masqué par défaut (display:none).

Il est stylé avec CSS et nous utilisons JavaScript pour le montrer (display: block).

Comment ça marche: Notez que l’attribut onclick est associé à l’élément p avec class = "flip". Lorsque l'utilisateur clique sur l'élément p, une fonction appelée myFunction () est exécutée, ce qui modifie le style du div avec id = "panel": aucun (masqué) pour afficher: bloc (visible).

Vous en apprendrez plus sur JavaScript dans notre didacticiel JavaScript.
