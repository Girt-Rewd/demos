Margin : auto

Attribuer une width à un élément block-level l'empêchera de s'étirer sur toute la largeur de son conteneur. Ensuite vous pouvez mettre la valeur auto aux propriétés margin-right et margin-left pour centrer horizontallement cet élément à l'intérieur de son conteneur. L'élément prendra la largeur que vous avez spécifié et l'espace restant sera séparé équitablement entre les deux marges. Le seul problème survient lorsque la fenêtre du navigateur est plus petite que la largeur de votre élément. Le navigateur résout cela en créant une barre de défilement (scrollbar) horizontale. Améliorons cette situation.

Margin : max-width

Utiliser max-width à la place de width améliorera la gestion des petites fenêtres. C'est très important lorsque l'on créé un site accessible sur mobile. Redimensionnez cette page pour voir ! Au fait, max-width est supporté par tous les navigateurs majeurs incluant IE7+, vous ne devez donc pas avoir peur de l'utiliser.

Display

la propriété "display" La propriété CSS display est la plus importante pour contrôler la mise en page. Chaque élément à une valeur display par défaut dépendante du type de l'élément. Les valeurs par défaut de la plupart des éléments sont block ou inline. Un élément block est souvent appelé un élément block-level. Position

La propriété de position CSS est polyvalente et puissante. Cela permet de définir ou de modifier la position d'un élément. Il a 4 valeurs possibles:
  1. static (default value)
  2. relative
  3. absolute
  4. fixed
Il est souvent utilisé aux côtés des 4 propriétés de coordonnées:
  1. left
  2. right
  3. top
  4. bottom

Relative

Lorsque la position est relative, un élément peut se déplacer en fonction de sa position actuelle.

Absolute

Lorsque la position est définie sur absolu, un élément peut se déplacer en fonction du premier ancêtre positionné.
La caractéristique d'un élément positionné est qu'il peut agir comme un point de référence pour ses éléments enfants.
Imaginons une hiérarchie simple:
Je suis dans une position relative.

Je suis dans une position absolute!

Je suis dans une position fixed!.

Fixed

Lorsque la position est mis à fixed, il agit comme absolu: vous pouvez définir les coordonnées gauche / droite et haut / bas. La seule différence est que le point de référence est la fenêtre. Cela signifie qu'un élément fixe ne défilera pas avec la page; il est fixé sur l'écran.

Float

Utilisation de float modifie non seulement l'élément sur lequel il est appliqué, mais modifie également ses ancêtres, ses frères et sœurs, ses descendants et les éléments suivants.
Le but d'appliquer float a un élément est de le pousser d'un côté et de le faire s'enrouler autour du texte.
